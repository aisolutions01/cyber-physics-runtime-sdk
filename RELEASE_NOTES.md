# v0.1.0-alpha

Initial research-stage release of the Cyber-Physics Runtime SDK.

Release date: 2026-05-28

## Overview

This release introduces the foundational runtime governance primitives for distributed operational stabilization research.

The SDK provides an early open infrastructure implementation focused on:

* Runtime entropy evaluation
* CFL boundary monitoring
* Governance state transitions
* Distributed telemetry observation
* Deterministic runtime stabilization experiments

The current release is intended exclusively for research and experimental infrastructure exploration.
